# Multi-Method-Options-Pricer
# Overview

This project is a Python based options pricing engine that compares multiple quantitative finance models for pricing European style options. It estimates historical volatility from SPY market data, prices options using analytical and numerical methods, and calculates option Greeks for risk analysis.


# Features

✅ Black-Scholes Model

✅ Black-Scholes-Merton (Continuous Dividend Yield)

✅ Cox-Ross-Rubinstein Binomial Tree

✅ Monte Carlo Simulation (European Options)

✅ Monte Carlo Simulation (Barrier Options)

✅ Historical Volatility Estimation

✅ Greeks Calculation
